# Sigilweaver OpenYXDB

A cross-platform C++ library and Python package for reading and writing YXDB files.

## What is YXDB?

YXDB is a row-oriented binary file format used by Alteryx Designer. Each file contains a schema (as XML metadata), followed by LZF-compressed blocks of records. Records store all fields together -- fixed-length fields packed with inline null flags, followed by variable-length data. A block index at the end of the file enables random access.

This library supports **E1 (non-AMP) YXDB files only**. Files generated by AMP (E2 format) are not supported.

## Features

- Read and write E1 YXDB files
- All 17 field types (Bool, Byte, Int16, Int32, Int64, FixedDecimal, Float, Double, String, WString, V_String, V_WString, Date, Time, DateTime, Blob, SpatialObj)
- Sequential and random-access record reading
- Cross-platform: Linux, macOS, Windows
- Python bindings (nanobind) with PyArrow, Pandas, and Polars support

## Python

```python
import openyxdb

# Read to PyArrow, Pandas, or Polars
table = openyxdb.read_arrow("data.yxdb")
df = openyxdb.read_pandas("data.yxdb")
df = openyxdb.read_polars("data.yxdb")

# Polars plugin: write back to YXDB
df.to_yxdb("output.yxdb")
```

## YXDB Limitations

- Spatial indexes are not supported. They are skipped on read and not created on write.
- Spatial objects are accessible only as raw blobs (SHP format).
- Little-endian architectures only.
